My Experience Creating a Portfolio Website

Last week, I had an interesting experience creating my own portfolio website for a school project. I felt excited and nervous because it was my first time making a complete website by myself. I wanted to create a simple but attractive website to show my profile, hobbies, skills, and projects.

First, I searched for website design inspiration on the internet. I looked at different layouts, colors, and animations to get ideas for my website. After that, I started creating the website using HTML, CSS, and Bootstrap. I made several sections such as Home, About Me, Skills, Projects, and Contact.

While creating the website, I faced some difficulties. The navigation bar and some buttons did not work properly. Besides that, the image layout looked messy and some text was not aligned correctly. I felt confused at first, but I kept trying to fix the problems by watching tutorials and searching for solutions on the internet.

After solving the problems, I added animations, pictures, icons, and personal information to make the website more interesting. I also changed some colors and fonts so the design looked cleaner and more modern. Then, I tested the website on my phone and computer to make sure the design looked good and responsive on different screen sizes.

Finally, I successfully finished my portfolio website. I felt very proud because I gained a lot of new experience in web design and coding. This experience taught me to be more patient, creative, and confident. I hope I can create even better websites in the future.
